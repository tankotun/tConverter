<h1>tConverter</h1>

<p>Supported Units: Weight, Length</p>
<br/><br/>
<h3>Require and start class.</h3>
Once, require file...
<pre><code>// Require
require "tConverter.class.php";
</code></pre>

<br/>
<pre>// Then, start the class.
<code>$tConverter = new tConverter();</code>
</pre>
